# Late Interaction evaluation v1

Retrieval-only labeled manifest for milestone 2.5. The sources are controlled
`demo-wave-v1` documents that have passed the real SQL/Qdrant publication and
governance lifecycle. Every run must preflight the identities against the
frozen source collection before benchmarking.

Negative cases intentionally name forbidden provenance so a retrieval leak is
observable without exposing document content in the report.
