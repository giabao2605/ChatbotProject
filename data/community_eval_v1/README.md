# Community summary evaluation v1

This is a target-locked evaluation template for roadmap milestone 2.8.

- `global` cases measure cross-document sensemaking.
- `local` cases guard ordinary retrieval quality.
- `relational` cases guard the approved GraphRAG path.
- The locked minimum global-answer gain is 10 percentage points.

The template is intentionally not executed while the milestone 2.7 graph gate
is blocked on independent edge review. No summary may be generated or served
from this manifest until `community-summary-readiness-v1.ready_for_generation`
is true. Source identities must be resolved against the same staging snapshot
immediately before a future baseline/candidate run.
